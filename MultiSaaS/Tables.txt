## TBL0051 (Table Master)

**Purpose:** Defines high-level metadata about all system tables including table names, descriptions, and configuration settings.
****

**Relationships:** Referenced by TBL0052 as the parent table for hierarchical metadata structure.

## TBL0052 (Table Details)

**Purpose:** Holds column-level details for each table including field names, data types, constraints, and validation rules.
****

**Relationships:** References TBL0051 via NUM002 (Table Master ID) to establish parent-child relationship.

## TBL0053 (Global Variable)

**Purpose:** Stores global key-value pairs used across modules for system-wide configuration and settings.

## TBL0054 (Utility Master)

**Purpose:** Defines utility metadata categories and classification schemes used throughout the system.
****

**Relationships:** Parent table used in conjunction with TAL0055 (Utility Details) for categorized data storage.

## TBL0055 (Utility Details)

**Purpose:** Holds specific values and configurations belonging to each Utility Master category.
****

**Relationships:** References TAL0054 as parent table to maintain hierarchical utility data structure.

## TBL0056 (Branding Settings)

**Purpose:** Contains branding and visual customization details including logos, color schemes, and styling configurations.
****

**Relationships:** Referenced by TBL0032 (Distributor Master) to apply distributor-specific branding.

## TBL0061 (Device Type)

**Purpose:** Lists and categorizes various types of devices supported by the AMS system including specifications and capabilities.

**Relationships:** Used in device-related views and reports but maintains loose coupling with device management tables.

## TBL0062 (Device Category)

**Purpose:** Groups devices by functional use-case, application domain, or operational category.
****

**Relationships:** Used in device metadata classification and likely referenced by device management systems.

## TBL0063 (Device Vendor)

**Purpose:** Stores comprehensive device vendor information including company details, contact information, and support data.

**Relationships:** Referenced by TBL0033 (Distributor Channel), TBL0064 (Device Protocols), and TBL0065 (Device Firmware).

## TBL0064 (Device Protocols)

**Purpose:** Defines communication protocols, data formats, and integration standards used by different device types.
****
